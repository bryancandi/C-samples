# The C Programming Language

A collection of C programs, samples, and exercises.  
Including standalone examples, utilities, and solutions inspired by **The C Programming Language** *(Kernighan & Ritchie)*.

## Contents
- [docs](./docs/) &mdash; Markdown notes about C / Unix topics
- [examples](./examples/) &mdash; Small C examples and utilities
- [games](./games/) &mdash; Small C games
- [graphics](./graphics/) &mdash; OpenGL experiments
- [k_and_r](./k_and_r/) &mdash; Classic / modernized exercises and code samples from K&R
- [msys2](./msys2/) &mdash; MSYS2 Setup on Windows x64 (MINGW64 & UCRT64)
- [tools](./tools/) &mdash; Portable utility programs that work across multiple platforms
- [unix_tools](./unix_tools/) &mdash; Unix-specific utility programs and Linux equivalents

## Building
Compile any program with:

```sh
gcc file.c -o file
```
